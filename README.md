# Basics of html and css
## Задание
1. Выбрать тематику сайта.
2. Сделать набросок структуры страниц (расположение заголовка, меню, основной области, боковых блоков, подвала и т.д. см. пример рис.1)
3. Разработать набор веб-страниц (HTML, CSS) выбранной тематики. Напол- нить страницу содержимым соответствующим предметной области выбранно- го сайта. На одной из страниц обязательно должна быть форма. Использовать блочную вёрстку.
4. Каждый набор должен включать не менее пяти страниц разного типа (ти- тульная, новости, поиск, карта сайта, каталог товаров и тому подобное).
